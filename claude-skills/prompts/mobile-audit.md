# Mobile Audit

Perform a comprehensive audit of the mobile application, evaluating:

- **Platform Compliance**: Adherence to iOS Human Interface Guidelines and Android Material Design
- **Performance**: App startup time, memory usage, battery consumption, network efficiency
- **Accessibility**: VoiceOver/TalkBack support, dynamic type, color contrast, touch target sizes
- **Offline Support**: Data caching, graceful degradation, sync conflict resolution
- **Security**: Secure storage, certificate pinning, biometric auth, data encryption at rest
- **Navigation**: Deep linking, back stack management, gesture handling
- **Responsiveness**: Layout adaptability across screen sizes, orientations, and foldables

## Output Format

Provide findings grouped by severity (Critical, High, Medium, Low) with:
1. Screen or component affected
2. Description of the issue
3. Recommended fix
4. Impact on user experience
