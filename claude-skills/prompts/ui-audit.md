# UI Audit

Perform a comprehensive audit of the user interface, evaluating:

- **Accessibility**: WCAG 2.1 AA compliance, ARIA labels, keyboard navigation, screen reader support
- **Consistency**: Design token usage, spacing, typography, color palette adherence
- **Responsiveness**: Breakpoint behavior, layout shifts, mobile/tablet/desktop rendering
- **Performance**: Largest Contentful Paint, Cumulative Layout Shift, image optimization, bundle size
- **Usability**: Form validation UX, error states, loading states, empty states
- **Cross-Browser**: Compatibility across Chrome, Firefox, Safari, Edge
- **Internationalization**: RTL support, text truncation, date/number formatting

## Output Format

Provide findings grouped by severity (Critical, High, Medium, Low) with:
1. Component or page affected
2. Screenshot or description of the issue
3. Recommended fix
4. Accessibility impact score
